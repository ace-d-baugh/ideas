# D'VINS - Disney Very Important Notification System

A cross-platform mobile application for iOS and Android that provides real-time wait times for attractions at Walt Disney World. The app retrieves data from the Queue-Times API (https://queue-times.com/) every 5 minutes, displaying attraction names, wait times, operational status (open/closed), and trending wait times (increasing/decreasing).

## Features

- **Real-time Wait Times**: Fetch and display current wait times for all Walt Disney World attractions
- **Trending Indicators**: Show wait time trends with green upward-right arrows (\u2197) for increasing and red downward-right arrows (\u2198) for decreasing
- **Operational Status**: Display open/closed status and downtime counters for non-operational attractions
- **Sorting & Filtering**: Sort attractions by park, favorites, alphabetical order, or wait times
- **Push Notifications**: Receive alerts for user-defined wait time thresholds or attraction reopenings
- **Cross-Platform**: Built with React Native for iOS and Android compatibility
- **Offline Support**: Cache last-known data for offline viewing

## Technical Stack

- **Frontend**: React Native (JavaScript)
- **Backend**: Node.js with Express
- **Database**: SQLite
- **Notifications**: Firebase Cloud Messaging
- **Version Control**: GitHub
- **Project Management**: Monday.com
